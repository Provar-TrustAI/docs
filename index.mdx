Provar Trust AI is an **agent evaluation platform** for the teams building, integrating, and operating production AI agents. Ingest real agent sessions, review and annotate turns, curate test datasets, define automated evaluators, run evaluations against new agent versions, and catch regressions — all before a change ships to customers. A built-in Trust Agent drives this loop for you in plain language, pausing for your approval before any write.

## The evaluation loop in 60 seconds

Trust AI is built around one core workflow: **ingest → annotate → curate → evaluate → compare → ship**.

1. **Ingest** real agent [Sessions](/concepts/sessions) from your connected runtime (AWS Bedrock AgentCore today, more connectors coming).
2. **Annotate** sessions with pass/fail, rubric scores, or freeform notes — human judgment becomes data.
3. **Curate** the most informative sessions into a [Dataset](/concepts/datasets) — a versioned, reproducible test bed.
4. **Define** [Evaluators](/concepts/evaluators) — LLM Judges for qualitative judgment, Structured Match for deterministic checks.
5. **Run** an [Evaluation](/concepts/evaluations) against a specific Agent Version — scored results per row.
6. **Compare** two Agent Versions on the same Dataset — the diff is the regression signal.

Then ingest the next batch of sessions and the loop continues. Read the full narrative at [The evaluation loop](/concepts/evaluation-loop). To run the whole loop by conversation with the Trust Agent, work through the capstone tutorial: [Evaluate with the Trust Agent](/tutorials/evaluate-with-the-trust-agent).

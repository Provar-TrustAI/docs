## What's Trust AI?

Provar Trust AI is an agent evaluation platform for the teams building, integrating, and operating production AI agents. QA Practitioners ingest real agent sessions, review and annotate turns, curate datasets, define LLM Judge and Structured Match evaluators, run evaluations against new agent versions, and compare results — all before a change ships to customers.

  **60-second mental model:** start with [The evaluation loop](/concepts/evaluation-loop), then read [Sessions](/concepts/sessions) → [Datasets](/concepts/datasets) → [Evaluators](/concepts/evaluators) → [Evaluations](/concepts/evaluations) in order. That's the workflow the product is built around.
</Tip>

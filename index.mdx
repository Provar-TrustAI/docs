Provar Trust AI is an **agent evaluation platform** for the teams building, integrating, and operating production AI agents. Connect your agent, define scenarios — persona-and-goal test cases with an authored expected output — run evaluations that score every persona, grade the real conversations your agent already had, and check the automated judges against your own verdicts — all before a change ships to customers. A built-in Trust Agent drives this loop for you in plain language, guiding you from an empty project to a first scored evaluation, and pausing for your approval before any write.

## The evaluation loop in 60 seconds

Trust AI is built around one core workflow: **connect → annotate → define → run → review → ship**.

1. **Connect** your agent — AWS Bedrock AgentCore, Salesforce Agentforce, or a Claude-managed agent — and ingest its real [Sessions](/concepts/sessions), including the agent's reasoning and model calls where the runtime provides them.
2. **Annotate** sessions with pass/fail verdicts, severity, and notes — human judgment becomes the ground truth the automated judges are measured against.
3. **Define** [Scenarios](/concepts/scenarios) — each one a named persona-and-goal test case with an authored expected output. Every project has one flat, searchable Scenarios list.
4. **Pick** [Evaluators](/concepts/evaluators) — LLM Judges for qualitative judgment, Structured Match for deterministic checks — or adopt the Trust Library's recommended evaluators and OWASP-aligned starter safety scenarios in one move, then tailor the rubrics in plain language.
5. **Run** an [Evaluation](/concepts/evaluations) against a specific Agent Version — scenarios simulate one conversation per persona, and captured sessions can be graded directly.
6. **Review** the dual-headline results — how often the agent passed, and how often the evaluators agreed with a human verdict — persona by persona, then refine and run again.

Then ingest the next batch of sessions and the loop continues. Read the full narrative at [The evaluation loop](/concepts/evaluation-loop). Starting from an empty project? Ask the Trust Agent — it takes you from connecting your agent, through an observe-only look at its behavior, to your first scored evaluation, all in one conversation. Work through the capstone tutorial: [Evaluate with the Trust Agent](/tutorials/evaluate-with-the-trust-agent).

Administering the workspace? Manage members, teams, and per-project Viewer and Editor access from **Workspace settings** — no code required.

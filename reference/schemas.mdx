These TypeScript-flavored shapes describe the objects you'll see on the wire. Field names are stable; optional fields and exact types may shift before v1.

## Session

```ts
type Session = {
  id: string;
  projectId: string;
  agentVersionId: string;
  createdAt: string; // ISO 8601
  turns: Turn[];
  metadata?: Record<string, string>;
};

type Turn = {
  id: string;
  role: "user" | "agent" | "tool";
  content: string;
  toolCalls?: ToolCall[];
  createdAt: string;
};

type ToolCall = {
  id: string;
  name: string;
  input: unknown;
  output?: unknown;
  latencyMs?: number;
};
```

## Dataset

```ts
type Dataset = {
  id: string;
  projectId: string;
  name: string;
  version: number;
  sessionIds: string[];
  createdAt: string;
  description?: string;
};
```

<Note>
  These shapes are hand-written for now. A future milestone will auto-generate this reference from the platform OpenAPI specification so the docs and the wire format never drift.
</Note>
